# Changelog

## 1.1.0 - Provozní a obchodní systém

- Přidáno admin notifikační centrum s badge v levém menu.
- Přidán CRM přehled firem, klientských účtů, inzerátů a financí.
- Dashboard rozšířen o provozní upozornění a odkaz na firmy.
- Admin UI dostalo jednotnější rytmus textů, sekcí, hlaviček a pracovních tabulek.
- Detail inzerátu byl posunut do redakčního pracovního rozhraní s financemi, reakcemi a historií.
